Golden Investments App

Dies ist eine Full-Stack-Anwendung, die die Kernfunktionalitäten und das Design von Airbnb nachbildet, jedoch unter dem Namen "Golden Investments" und mit dem Hauptakzent in Babyblau/Telegram Blau (((\>($\approx \#2 \text{EA} 7 \text{EE}$)</))).

Architektur

/client: Frontend-Anwendung (React/JSX-Struktur).

/server: Backend-API (Node.js/Express-Struktur) zur Verwaltung von Unterkünften, Entdeckungen, Services und Benutzerauthentifizierung.

/shared: Geteilte Konstanten und Datentyp-Definitionen.

Features (Basierend auf Airbnb-Screenshots)

Suchfunktion: Unterkunft, Entdeckungen, Services.

Kartenansichten: Anzeigen von Unterkünften ("Gäste-Favoriten") und Entdeckungen ("Beliebte Entdeckungen").

Wunschlisten: Speichern von Favoriten (Login erforderlich).

Authentifizierung: Einloggen/Registrieren (Telefonnummer, Google, Apple, E-Mail).

Services: Anzeige von professionellen Dienstleistungen (z.B. Fotografie, Köch:innen) mit Qualitätsüberprüfung.

Setup

Backend:

Navigieren Sie zu /server.

Installieren Sie die Abhängigkeiten (npm install).

Starten Sie den Server (node server.js).

Frontend:

Navigieren Sie zu /client.

Installieren Sie die Abhängigkeiten (abhängig vom gewählten Framework, z.B. npm install).

Starten Sie die Anwendung
